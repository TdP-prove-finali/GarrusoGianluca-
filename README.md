# Tesi Tecniche di Programmazione

## Applicazione per l'analisi e l'ottimizzazione di reti musicali tramite grafi

### Istruzioni per l'installazione e l'utilizzo dell'applicazione:

1. Effettuare il fork del progetto al seguente link: https://github.com/TdP-prove-finali/GarrusoGianluca-
2. Clonare il progetto dal repository all'interno del proprio ambiente di programmazione.
3. Installare le librerie elencate nel file Requirements.txt.
4. Configurare il database Chinook seguendo le indicazioni presenti nella cartella `database`.
5. Eseguire la classe `main.py` per avviare l'applicazione.
